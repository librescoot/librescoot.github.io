Navigation

Karten und Turn-by-Turn-Routenführung direkt auf dem Roller-Display. Das Feature, das immer versprochen, aber nie geliefert wurde.

Wie es funktioniert

Librescoot betreibt einen kompletten Navigations-Stack auf dem Dashboard-Rechner. Kartenkacheln werden aus einer lokalen Datenbank gerendert, Routen berechnet eine lokale Instanz von Valhalla. Keine Internetverbindung nötig.

  • Offline-Kartendarstellung mit vorgerenderten OpenStreetMap-Kacheln in einer .mbtiles-Datei
  • Offline-Routing mit Valhalla lokal auf dem Dashboard-Rechner
  • Turn-by-Turn-Anweisungen während der Fahrt auf dem 480x480-Display
  • Gespeicherte Orte für schnellen Zugriff auf häufige Ziele

Online-Option

Wenn du eine SIM-Karte eingelegt hast, kannst du auch Online-Kartenkacheln und einen Remote-Routing-Server nutzen. Kartendarstellung und Routing lassen sich unabhängig voneinander zwischen Offline- und Online-Modus umschalten.

Erste Schritte

Karten- und Routing-Kacheln für deine Region herunterladen, per USB auf den Roller übertragen – fertig.

Abdeckung

Vorgefertigte Kachelpakete gibt es für alle 16 deutschen Bundesländer. Für andere Regionen erzeugst du dir die Kacheln aus jedem beliebigen OpenStreetMap-Auszug mit maptool (Anzeige) oder dem Standard-Valhalla-Build-Prozess (Routing).
