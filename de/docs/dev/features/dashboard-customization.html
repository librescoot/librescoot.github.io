Dashboard anpassen

Light- und Dark-Mode, konfigurierbare Anzeige und endlich ein Weg, die kaputte Uhr zu reparieren oder auszublenden.

Display-Themes

Das Dashboard unterstützt Light- und Dark-Theme, umschaltbar im Einstellungsmenü. Die Helligkeit passt sich automatisch an den Umgebungslichtsensor an.

Konfigurierbare Elemente

Was auf deinem Dashboard erscheint, bestimmst du:

  • Uhr ein-/ausblenden: blende die Uhr aus, wenn sie die falsche Zeit anzeigt (das macht sie nämlich offline, weil keine RTC verbaut ist)
  • Akkuanzeige: Prozent oder Reichweite
  • Status-Icons: einzelne Indikatoren gezielt ein- oder ausschalten
  • Kartendarstellung: Vektor- oder Raster-Kacheln

Einstellungen

Dashboard-Einstellungen erreichst du über das Bildschirmmenü (doppelt auf die linke Bremse tippen) oder per lsc-CLI:

lsc settings list                          # Alle Einstellungen anzeigen
lsc settings set dashboard.map.type offline  # Auf Offline-Karten umschalten
lsc settings set dashboard.theme dark        # Dark-Theme
