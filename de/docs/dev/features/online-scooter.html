Roller online bringen

Eine SIM-Karte schaltet Fernzugriff, Online-Karten und Over-the-Air-Updates frei. Der komplette Weg: Hardware, Modem-Konfiguration, OTA-Einstellungen.

Optional, nicht zwingend

Librescoot läuft auch komplett ohne SIM. Was offline geht, steht in Funktioniert komplett offline. Alles unten dreht sich darum, Mobilfunk obendrauf zu legen.

1. SIM-Karte einbauen

Das 4G-Modem auf der MDB braucht eine Mini-SIM (2FF) in einem aufgelöteten SIM-Halter. Beim unu Scooter Pro ab Werk ist der Halter meist nicht bestückt – auf den meisten Rollern ist das also Lötarbeit.

  • Footwell öffnen, alle Akkus abklemmen (CB und AUX) und die MDB ausbauen. Die komplette Prozedur steht in der Installationsanleitung.
  • MDB-Gehäuse öffnen (Torx T10) und das SIM-Halter-Footprint neben dem SIM7100E-Modem suchen.
  • Einen Mini-SIM-Halter auf die Pads löten. Jeder Standard-6-Pin-Halter (Push-Push oder Klapp) für 2FF-Karten passt.
  • SIM einlegen, Gehäuse zu, MDB wieder einbauen, Akkus in umgekehrter Reihenfolge anschließen (erst AUX, dann CB, dann Hauptakkus).

Wenn möglich, eine Daten- oder M2M-SIM verwenden – Sprache und SMS werden nicht genutzt. Formfaktor ist Mini-SIM (2FF); Micro- oder Nano-SIM nur im Notfall zurechtschneiden.

Vor dem Öffnen der MDB alle Akkus abklemmen

Gleiche Reihenfolge wie bei der Installation: Hauptakkus raus, CB ab, dann AUX ab. Beim Zusammenbau in umgekehrter Reihenfolge.

2. APN setzen

Das Modem braucht den APN deines Providers. Per SSH auf den Roller, dann mit lsc setzen:

lsc set cellular.apn internet

Statt internet den APN deines Providers eintragen – in Deutschland gängig: internet (Telekom, o2), internet.eplus.de (E-Plus/o2-Altbestand), web.vodafone.de (Vodafone). Steht meist in der Onboarding-Mail oder auf der Provider-Website.

Die Änderung greift sofort. modem-service übernimmt den neuen Wert und konfiguriert den Bearer neu.

3. SIM-PIN setzen (falls nötig)

Hat die SIM eine PIN-Sperre, die PIN hinterlegen, damit das Modem die Karte beim Boot entsperren kann:

lsc set cellular.sim-pin 1234

Schritt überspringen, wenn deine SIM keine PIN-Sperre hat. Ein leerer Wert lässt die SIM, wie sie ist. Die PIN landet im lokalen Redis-Settings-Hash und verlässt den Roller nicht.

Modem-Status prüfen mit:

lsc status

4. OTA-Updates konfigurieren

Sobald der Roller online ist, willst du Over-the-Air-Updates. Drei Einstellungen jeweils für MDB und DBC, plus eine fürs Orchestrieren.

Channel

stable für Produktion, testing für frühen Zugriff, nightly für Entwicklungs-Builds. Mit dem Shortcut beide auf einmal setzen:

lsc ota channel stable

Methode

delta lädt nur die geänderten Blöcke (klein, schnell). full lädt das komplette Image. Delta ist Default und auf einer kostenpflichtigen SIM die richtige Wahl.

lsc set updates.mdb.method delta updates.dbc.method delta

Check-Intervall

Wie oft der Roller nach neuen Releases fragt. Go-Duration-String – 24h für täglich, 6h für den Default, 0 deaktiviert die automatischen Checks. Achtung: 1d ist kein gültiges Go-Duration-Format, also 24h nehmen.

lsc set updates.mdb.check-interval 24h updates.dbc.check-interval 24h

MDB orchestriert DBC

Die DBC (Dashboard) ist die meiste Zeit aus, kann also selbst nicht nach Updates schauen. Mit Orchestrierung an schaltet die MDB die DBC ein, sobald ein neueres DBC-Release verfügbar ist, und stößt dort einen Update-Check an.

lsc set updates.mdb.orchestrate-dbc true

Empfohlene Defaults

Für die meisten Fahrer:innen:

lsc ota channel stable
lsc set \
  updates.mdb.method delta \
  updates.dbc.method delta \
  updates.mdb.check-interval 24h \
  updates.dbc.check-interval 24h \
  updates.mdb.orchestrate-dbc true

5. Funktioniert's?

Modem- und OTA-Status prüfen:

lsc status        # Mobilfunk-Registrierung, Signal, IP
lsc ota status    # aktuelle OTA-Konfiguration und letzter Check
lsc ota check     # jetzt nach Updates suchen

Wenn lsc ota check den aktuellen Channel und einen Last-Check-Zeitstempel meldet, redet der Roller mit dem Release-Server.

Cloud-Uplink (optional)

OTA-Updates funktionieren, sobald der Roller Internet hat – Cloud-Account ist nicht nötig. Wer Status, Position und Akkustand zusätzlich vom Handy oder Browser sehen will, meldet den Roller bei einer Cloud an:

  • Sunshine Cloud – der Community-Uplink. Privat kostenlos.
  • Selbst hosten mit uplink-server.
