Roller-Alarmanlage

Eingebaute Alarmanlage, die auslöst, wenn sich jemand an deinem Roller zu schaffen macht.

Wie es funktioniert

Die Alarmanlage nutzt den BMX055-Beschleunigungssensor im MDB, um Bewegung zu erkennen. Wenn der Roller abgeschlossen und die Alarmanlage scharf ist, löst jede Bewegung oder Manipulation eine zweistufige Reaktion aus:

  • Stufe 1 (leichte Bewegung): Warnblinker blinkt als Warnung
  • Stufe 2 (anhaltende Bewegung): Warnblinker blinkt und die Hupe ertönt

Die Alarmanlage schärft sich automatisch, sobald du den Roller abschließt (wenn aktiviert). Sie wird entschärft, sobald du mit Keycard oder per BLE aufsperrst.

Konfiguration

Die Alarmanlage steuerst du über die lsc-CLI:

lsc alarm arm          # Alarmanlage scharfschalten
lsc alarm disarm       # Alarmanlage entschärfen
lsc alarm status       # Aktuellen Alarm-Status anzeigen

lsc settings set alarm.enabled true    # Beim Abschließen automatisch scharfschalten
lsc settings set alarm.honk true       # Hupe in Stufe 2 aktivieren
lsc settings set alarm.duration 30     # Alarmdauer in Sekunden

Warum das wichtig ist

Die Originalfirmware hatte gar keine Alarmfunktion. Librescoot bringt sie als eingebautes Feature mit – ohne externe Hardware, ohne Abo.
