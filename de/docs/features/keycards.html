Eigene Keycards

Neue NFC-Keycards selbst anlernen, so viele du willst.

Wie es funktioniert

Librescoot nutzt NFC-Keycards zum Auf- und Abschließen. Karte an den Leser am Dashboard halten – Roller auf. Jede NFC-Karte mit UID funktioniert.

In der Originalfirmware waren Keycards fest auf den Roller gebunden und konnten vom Nutzer nicht geändert werden. Und seit unu GmbH weg ist, kann das auch niemand mehr ändern. Librescoot gibt dir die Kontrolle zurück:

  • Beliebige NFC-Karten als Keycard nutzen
  • Keine Obergrenze bei der Anzahl der Keycards
  • Master-Keycards können neue Karten ohne CLI-Zugriff anlernen
  • Export und Import der Keycard-Liste für Backups

Keycards verwalten

Mit der lsc-CLI verwaltest du deine Keycards:

lsc keycard list               # Alle autorisierten Keycards anzeigen
lsc keycard add <uid>          # Keycard hinzufügen
lsc keycard remove <uid>       # Keycard entfernen
lsc keycard add-master <uid>   # Master-Keycard hinzufügen
lsc keycard export cards.json  # Für Backup exportieren
lsc keycard import cards.json  # Aus Backup importieren

Neue Keycards kannst du auch direkt über eine Master-Keycard am Dashboard anlernen – kein SSH-Zugang nötig.
