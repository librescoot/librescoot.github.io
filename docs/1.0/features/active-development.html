Active Development

New features and fixes ship regularly, built by people who ride these scooters.

Community-driven

Librescoot is built by its users. The people writing the code ride these scooters daily. Features get built because someone needed them, not because a product manager scheduled them.

  • Regular releases on testing and nightly channels
  • Open development with public issues, PRs, and Discord discussions

Release channels

ChannelDescription
StableRecommended for most users. Thoroughly tested releases.
TestingEarly access to new features with basic validation.
NightlyLatest builds from main branches. For developers.

Get involved

The codebase is mostly Go (services) and Qt/C++ (dashboard UI). Each service is its own repository with standard build tooling.
