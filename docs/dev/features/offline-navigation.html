Navigation

Maps and turn-by-turn routing directly on the scooter display. The feature that was promised but never shipped.

How it works

Librescoot runs a full navigation stack on the scooter's dashboard computer. Map tiles are rendered from a local database, and routes are calculated by a local Valhalla routing engine. No internet connection required.

  • Offline map display using pre-rendered OpenStreetMap tiles stored in an .mbtiles file
  • Offline routing with Valhalla running locally on the dashboard computer
  • Turn-by-turn directions shown on the 480x480 display while riding
  • Saved locations for quick access to frequent destinations

Online option

If you have a SIM card installed, you can also use online map tiles and a remote routing server. Both map display and routing can be independently switched between offline and online mode.

Getting started

Download map and routing tiles for your region, transfer them to the scooter via USB, and you're set.

Coverage

Pre-built tile packages are available for all 16 German states. For other regions, generate tiles from any OpenStreetMap extract using maptool (display) or the standard Valhalla build process (routing).
