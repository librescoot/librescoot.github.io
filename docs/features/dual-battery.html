Dual Battery Support

Use both battery slots for more range.

What it does

LibreScoot supports running two batteries simultaneously. The battery service monitors both via NFC and manages them together. Both batteries discharge in parallel, giving you roughly double the range of a single battery.

The scooter works fine with just one battery — this is for users who want to add a second one.

Hardware mod required

The stock scooter only has one battery slot wired up. To use the second slot, you need to add the connector and cable yourself. This is a hardware modification — you'll need to open up the scooter and route the wiring to the second battery bay.

Software side

Once the hardware is in place, LibreScoot handles the rest. The battery service auto-detects when a second battery is present and starts managing both. No configuration needed.
