Handbook

The Librescoot rider handbook is currently available in German only.

An English translation is planned but not yet available. In the meantime:

  • Read the German version — most browsers will offer to translate it automatically.
  • Browse the developer and installation documentation, which is available in English.
  • Ask in the Discord if you have a specific question about using the scooter.

Want to help translate?

If you'd like to contribute an English translation, the source files live in /de/handbook/. Open a pull request or get in touch on Discord.
